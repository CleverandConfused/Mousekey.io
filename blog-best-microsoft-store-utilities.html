7 Low-Key Microsoft Store Utilities That Are Actually Worth Installing

The Microsoft Store has a reputation problem. Most people think of it as a place for games and apps they'll never use. But there's a growing collection of lightweight utilities that solve specific problems Windows doesn't handle on its own. These are the ones that earned a permanent spot on my machine.

Every tool on this list is available directly from the Microsoft Store, which means auto-updates, easy uninstall, and no hunting for download links on random websites. Most are free. The ones that aren't are cheap enough that you'll forget you paid for them.

1. MouseKey

Free (Pro $4.99) · Microsoft Store

This is ours, so take the recommendation accordingly. MouseKey remaps mouse buttons and adds up to 6 actions per button using click cadences. Single click your side button for copy, double click for paste, triple click for undo, and so on up to 5x clicks plus a 2-second hold. Works with any mouse from any brand, runs fully offline, and collects zero data.

It also includes a timed macro recorder, automatic profile switching per app, typed string playback for commands like git status, and system controls for volume, brightness, pointer speed, and scroll speed. The free version covers most use cases. Pro unlocks unlimited profiles and extra settings.

2. Microsoft PC Manager

Free · Microsoft Store

This is a first-party Microsoft app that somehow most people don't know about. It's an all-in-one utility for system health: one-click memory cleanup, startup app management, storage cleanup, pop-up blocker for annoying app notifications, and a system monitor that shows CPU, memory, and network usage in a clean dashboard.

Think of it as a lighter, more focused version of CCleaner, except it's made by Microsoft and doesn't try to upsell you constantly. The boost button alone is worth the install. It kills unnecessary background processes and frees up RAM in about two seconds. If you've ever felt your PC slowing down after a few hours of use, PC Manager addresses that directly.

3. EarTrumpet

Free, open source · Microsoft Store

Windows has a volume mixer, but it's buried in settings and awkward to get to. EarTrumpet replaces your system tray volume icon with a proper per-app volume control. Click it and you see every app that's currently playing audio, each with its own volume slider. You can set Spotify to 40% while keeping your game at 80% and Discord at 60%, all from a single click in the tray.

It also lets you route apps to different audio output devices. Send your music to your speakers while keeping your meeting on your headset. This is something Windows technically supports but makes unnecessarily hard to configure. EarTrumpet makes it obvious.

4. DevToys

Free, open source · Microsoft Store

If you're a developer, DevToys is a toolbox of small converters and formatters that you'd otherwise use random websites for. JSON formatter, Base64 encoder/decoder, JWT decoder, URL encoder, hash generator, UUID generator, regex tester, color picker, text diff, and more. Everything runs locally. No data leaves your machine, which matters when you're pasting API keys or tokens into a formatter.

The interface is clean and each tool opens in its own tab. It's the kind of app you open once a day for 30 seconds, but it saves you from pasting sensitive strings into sketchy online tools. Non-developers probably don't need this one, but if you write code, it belongs on your taskbar.

5. OneCommander

Free (Pro available) · Microsoft Store

File Explorer works fine until you need to do anything involving two folders at once. OneCommander is a dual-pane file manager with tabs, breadcrumb navigation, built-in previews for images and documents, and a dark mode that actually looks good. You can drag files between panes, use column view like macOS Finder, and customize the layout to fit how you actually work with files.

The free version is fully functional for daily use. It doesn't try to replace File Explorer system-wide. It just gives you a better option when you need one. Especially useful if you regularly move files between folders or work with large directory structures.

6. FxSound

Free · Microsoft Store

Windows still doesn't have a built-in system equalizer. FxSound fixes that. It works at the system level, meaning it improves audio across everything: games, YouTube, Spotify, video calls, all of it. Out of the box you get presets for music, movies, voice, and bass boost. You can also create your own EQ curves and save them as custom presets.

It's particularly useful if you're using laptop speakers or budget headphones where the default sound is flat or tinny. A quick bass boost or voice clarity preset makes a noticeable difference without needing to touch hardware settings.

7. Scolect

Free · Microsoft Store

Windows has no built-in screen time tracking. Scolect fills that gap. It monitors how much time you spend in each application and generates reports you can export to Excel. You can create custom categories, flag apps as productive or unproductive, set time limits with alerts, and use a built-in focus mode with a pomodoro timer.

Everything stays local on your device. No account, no data sent anywhere. If you've ever wondered where your workday went, Scolect gives you the answer without the surveillance-tool vibe of enterprise monitoring software.

Why the Microsoft Store specifically

All of these tools are also available as direct downloads from their developers' websites. So why install them from the Store? Three practical reasons.

Auto-updates. Store apps update silently in the background. You don't have to check for new versions, download installers, or click through update wizards. The app stays current without you thinking about it.

Clean uninstall. Store apps uninstall cleanly through Windows Settings. No leftover registry entries, no orphaned folders, no remnants scattered across your system. If you decide you don't need one of these tools anymore, it's gone in two clicks.

Verified source. You know you're getting the real app from the real developer. No bundled adware, no wrapper installers, no "custom install" checkboxes that sneak in toolbars. The Store isn't perfect, but it eliminates the most common download risks.
